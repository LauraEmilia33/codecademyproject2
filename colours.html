Colours

Here you can find a range of my favorite color picker tools and how I get inspiration for my colors. I'll also be showing some of my favorite color combinations from these tools.

1. ColorZilla

ColorZilla is a must for all chrome users and coders. With this tool you can just keep it as an extension and take a color from any page whenever you want. So handy.

2. Colorhunt

Color hunt is my favorite because it allows you to see how different colour combinations work and you can easily select the one you like by simply coping the CSS code. Very easy to use. You can also save your favorite color combinations just in case you want to use them later! Did I mention it's also free to use?

3. Coolors

Coolors is a very similar website to Color hunt, however here you can upload an image and get the colors from that image, which I think can be really handy especially when you are creating a brand color combination or you are particullarly inspired by an image. Also very easy to use and there are many ways to export the colors, not just CSS!
